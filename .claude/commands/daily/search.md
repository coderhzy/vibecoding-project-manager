# /daily:search - 搜索历史日志

搜索历史开发日志中的内容。

## 使用方式

```
/daily:search "关键词"
/daily:search "登录" --from 2024-01-01
/daily:search "bug" --last 7d
```

## 参数

| 参数 | 说明 |
|------|------|
| 关键词 | 要搜索的内容 |
| --from | 起始日期 |
| --to | 结束日期 |
| --last | 最近 N 天 (7d, 30d) |

## 执行步骤

1. 解析搜索参数
2. 遍历 `.claude/daily/` 目录
3. 在每个日志文件中搜索关键词
4. 返回匹配结果及上下文

## 输出格式

```
搜索结果：找到 5 条匹配

📅 2024-01-10
  [需求记录] ...实现用户**登录**功能...
  [执行记录] ...完成**登录**接口...

📅 2024-01-08
  [TODO清单] ...修复**登录**bug...
```
